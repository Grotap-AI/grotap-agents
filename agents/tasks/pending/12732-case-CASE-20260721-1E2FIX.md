Backend _VALID_LANE_CHOICES = {auto,team1,team2,team3,team4,local} = the lane.id. Send lane.id, not lane.lane. Because the wrong value (B/C/A) is stored, activeTeamId (lane.id === activeTeamId) never matches, so the active-lane highlight also breaks. Scope: ClaudeAppPage.tsx createSession() signature + onLaneCreate handler + the POST body only. Run tsc.
