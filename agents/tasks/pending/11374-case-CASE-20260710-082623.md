# Task: Feature: Platform — Create ONE new file: scripts/llm_cost_report.py — a read-only CLI report of LLM

## Context
No additional context provided.

## Requirements
Create ONE new file: scripts/llm_cost_report.py — a read-only CLI report of
LLM spend from the llm_usage_log table (control-plane Postgres).

WHY: operators currently hand-write these SQL queries every time spend questions come up.

BEHAVIOR:
- argparse with one option: --days N (int, default 14).
- Section 1 "Spend by service/model": SELECT service, model, count(*) AS calls,
  ROUND(SUM(cost_usd)::numeric, 4) AS total_usd, ROUND(AVG(cost_usd)::numeric, 4) AS avg_usd
  FROM llm_usage_log WHERE ts > now() - ($1 || ' days')::interval
  GROUP BY 1, 2 ORDER BY total_usd DESC NULLS LAST.
- Section 2 "Daily spend": SELECT ts::date AS day, ROUND(SUM(cost_usd)::numeric, 2) AS usd,
  count(*) AS calls FROM llm_usage_log WHERE ts > now() - ($1 || ' days')::interval
  GROUP BY 1 ORDER BY 1.
- Print a grand-total line at the end (sum of cost_usd over the window).
- Use asyncio.run(main()) like scripts/db.py does. Exit code 0 on success, 2 when
  DATABASE_URL is unset (print a clear error to stderr, mirroring scripts/db.py).

SCHEMA FACTS (do not guess others): llm_usage_log has columns ts (timestamptz), service (text),
model (text), lane (text), cost_usd (numeric/float, nullable), ok (bool), case_id (text).

GUARDRAILS (STRICT):
- Create ONLY the one new file named above. Do NOT modify, delete, or reformat ANY existing file.
- Read-only: every SQL statement must be a SELECT. No INSERT/UPDATE/DELETE/DDL anywhere.
- Connect with asyncpg using os.environ["DATABASE_URL"] (injected by Doppler at runtime).
  Never hardcode a connection string or any secret. Follow the connection pattern in
  scripts/db.py (asyncpg.connect, try/finally close).
- No new dependencies: standard library + asyncpg only (asyncpg is already used by scripts/db.py).
- Keep the script under ~150 lines. Output is plain text to stdout (aligned columns or
  pipe-delimited rows are both fine).
- The file must pass: python3 -m py_compile scripts/<filename>.
